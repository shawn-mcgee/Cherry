//Default Keyboard
[Tab] Toggle Brush Mode
[1] Tile Mode
[2] Wall Mode
[=] Increment Camera Zoom
[-] Decrement Camera Zoom
[W, ↑] Translate Camera Up   
[S, ↓] Translate Camera Down
[A, ←] Translate Camera Left
[D, →] Translate Camera Right
[Q] Last Brush
[E] Next Brush

//Default Mouse
[Mouse 1] Place
[Mouse 3] Break
[Mouse 2] Drag Camera
[Wheel  ] Zoom Camera

//Ctrl + Keyboard
[Ctrl + N] New
[Ctrl + O] Open
[Ctrl + S] Save

//Alt + Keyboard
[Alt + ~] Show/Hide Grid
[Alt + 1] Show/Hide Tiles
[Alt + 2] Show/Hide Walls

//Shift + Keyboard
[Shift + =] Increment Room Size
[Shift + -] Decrement Room Size
[Shift + W, Shift + ↑] Translate Cells Up
[Shift + S, Shift + ↓] Translate Cells Down
[Shift + A, Shift + ←] Translate Cells Left
[Shift + D, Shift + →] Translate Cells Right
[Shift + Q] Rotate Cells Counter Clockwise
[Shift + E] Rotate Cells Clockwise
[Shift + Z] Mirror Cells on Y Axis
[shift + X] Mirror Cells on X Axis
